Our projects

Xash3D FWGS

A Xash3D engine fork with many new features: high portability, advanced multiplayer, touchscreen & gamepad support and much more.

CS16Client

A Counter-Strike 1.6 Android port using Xash3D FWGS engine, rewritten CS1.6 client library and ReGameDLL_CS

MainUI C++

A GUI library for Xash3D, written in C++. Includes window system, powerful font rendering and dynamic menus. Supports both original Xash3D and Xash3D FWGS.

NanoGL

Super fast OpenGL-to-OpenGLES translator. Originally written by Olli Hinkka.

BSLauncher
